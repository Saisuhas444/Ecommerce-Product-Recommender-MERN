# E-Commerce Product Recommender (MERN)

A placement-focused MERN Stack project.

## Stack
Frontend: React, Redux Toolkit, React Router, Axios, Tailwind CSS
Backend: Node.js, Express.js, MongoDB Atlas, Mongoose, JWT, bcryptjs

## Features
- Authentication
- Product Catalog
- Search & Filters
- Wishlist
- Cart
- Orders
- Product Recommendation Engine
- Admin Dashboard

## Recommendation Logic
Category Match + User Activity + Ratings + Trending Score
